# Clicky Game

## Description
Test your memory with this Star Wars clicky game! When the page loads, start by clicking any image. Don't click the same image more than once. The images will randomize after every click. Click them all just once and you win! Check out the live game <a href="https://blakeneely.github.io/clicky-game/">here</a>.

## Technologies
    - HTML, CSS
    - Bootstrap
    - React.js
